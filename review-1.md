# TaskManager API

## ¿Qué es mi producto y para qué sirve?

TaskManager API es una herramienta digital que te permite organizar y gestionar tus tareas diarias de manera sencilla y eficiente. Funciona como un asistente personal que guarda, organiza y te ayuda a dar seguimiento a todas las actividades que necesitas completar.

Es ideal para cualquier persona o equipo que quiera mantener un control ordenado de sus pendientes, ya sea para uso personal, académico o profesional.

## ¿Cuáles son las funcionalidades más importantes y por qué los usuarios las usarían?

### Ver todas tus tareas
Puedes consultar la lista completa de todas tus tareas en cualquier momento. Esto te permite tener una visión general de todo lo que tienes pendiente.

### Buscar una tarea específica
Si tienes muchas tareas, puedes buscar una en particular sin tener que revisar toda la lista. Ahorra tiempo y te lleva directamente a lo que necesitas.

### Filtrar por estado
Puedes ver solo las tareas completadas o solo las pendientes. Esto es útil para enfocarte en lo que falta por hacer o para revisar lo que ya lograste.

### Crear nuevas tareas
Agrega nuevas tareas de forma rápida y sencilla. Solo necesitas escribir qué necesitas hacer y listo.

### Actualizar tareas
¿Cambió algo en tu tarea? Puedes modificar los detalles o marcarla como completada cuando la termines.

### Eliminar tareas
Cuando una tarea ya no es necesaria, puedes eliminarla para mantener tu lista limpia y organizada.

## ¿Qué tecnologías usé y por qué?

### Node.js
Es el motor que hace funcionar la aplicación. Permite que el sistema sea rápido y pueda atender muchas solicitudes al mismo tiempo sin problemas.

### Express
Es como el organizador interno de la aplicación. Ayuda a que todas las funciones (crear, ver, editar, eliminar tareas) estén bien estructuradas y funcionen correctamente.

### Nodemon
Es una herramienta que facilita el desarrollo. Permite que los cambios se apliquen automáticamente sin necesidad de reiniciar la aplicación manualmente.
